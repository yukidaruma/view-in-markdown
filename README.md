## Development

Use `COPY_MARKDOWN_SOURCE=1` to build/develop "Copy Markdown Source" version.

```
# View Markdown Source version:
bun dev
bun zip

# Copy Markdown Source version:
COPY_MARKDOWN_SOURCE=1 bun dev
COPY_MARKDOWN_SOURCE=1 bun zip
```
